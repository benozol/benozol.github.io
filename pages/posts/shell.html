Shell tricks

Zsh, your IDE

During development, some commands are used frequently, for example compilation or testing. So far, you would need to re-type them, or search in the history. But in an IDE there would be keyboard shortcuts for that, so why not in your shell? Use the following lines to put your preferred commands in variable $MYMAKE1 and call them just by pressing alt-1. 

# My make, a global entry for running something by pressing alt-m
mymake1() {
    [ -n "$MYMAKE1" ] && { BUFFER="$MYMAKE1"; zle accept-line }
}
zle -N mymake1
bindkey '^[1' mymake1

This can be repeated by substituting 1 for commands for more shortcuts. (Or can bindkey commands be parameterised to avoid the repetition?) And environment parameters aren't the only or best way to save the commands: these could likewise be stored in (per-directory) config files.
